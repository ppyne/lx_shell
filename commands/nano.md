# nano

Minimal nano-style editor built on the same editor core as `vi`.

## Usage

```
nano [path]
```

## Shortcuts

- `Ctrl+O` write (prompts for file name)
- `Ctrl+X` exit (press twice to discard unsaved changes)
- `Ctrl+W` search
- `Ctrl+K` cut line
- `Ctrl+U` paste line
- `Ctrl+G` help hint in status bar
